README file of R scripts for USMCP manuscripts
1.	System requirements
platform       x86_64-w64-mingw32               
arch           x86_64                           
os             mingw32                          
crt            ucrt                             
system         x86_64, mingw32   

2.	Installation guide
There is no new software developed for this manuscript. Standard installation guide for R can be found at https://cran.r-project.org/bin/windows/base/.

3.	Demo
The data input for running the scripts had been provided as Supplemental Data. The expected output had been provided as figures and supplemental figures. Expected run time for all scripts are within minutes on a normal desktop computer.

4.	Instructions for use
The .R scripts can be executed in the RStudio environment. General instructions on how to run .R scripts can be found at RStudio User Guide https://docs.posit.co/ide/user/ide/guide/code/execution.html
